# Dibal Abi Moussa - Fitness Trainer Portfolio

A professional, modern portfolio website showcasing the expertise, certifications, and client transformations of a certified fitness coach specializing in strength conditioning, body recomposition, and prenatal/postnatal training.

---

## 🎯 Project Importance

This portfolio serves as a **digital storefront** for a fitness professional, designed to:

- **Establish credibility** through showcasing international certifications (EREPS Level 5, ACE, NCSF)
- **Attract high-quality clients** by demonstrating expertise, training methodology, and proven results
- **Highlight specializations** including body recomposition, post-rehabilitation, prenatal/postnatal training, and sports nutrition
- **Showcase success stories** with before/after client transformations and testimonials
- **Enable direct contact** for consultation and training inquiries
- **Build professional presence** in the competitive fitness coaching industry with a modern, responsive design

In today's digital landscape, a well-designed portfolio is essential for service professionals to compete, build trust, and convert potential clients into loyal customers.

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/agnesabimoussa/Fitness-Trainer-Portfolio.git
   cd Fitness-Trainer-Portfolio
   ```

2. **Navigate to project directory**
   ```bash
   cd project
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

### Running the Project

**Development Mode** (with hot reload)
```bash
npm start
```
The app will open at `http://localhost:3000`

**Build for Production**
```bash
npm build
```

**Run Tests**
```bash
npm test
```
